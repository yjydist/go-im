## 当前仓库速览（接手前 1 分钟）

- [事实] 单仓库 Go 项目（`go.mod`）。
- [事实] 三个服务入口：
  - `cmd/api/main.go`（HTTP API）
  - `cmd/ws/main.go`（WebSocket 网关 + 内部推送入口）
  - `cmd/push/main.go`（Kafka 消费 + 推送）
- [事实] 核心外部依赖：MySQL / Redis / Kafka（`config/go-im.yaml`、`deploy/docker-compose.yaml`）。
- [事实] 本地启动方式：`make env-up` + 分别 `make run-api|run-ws|run-push`（`Makefile`）。
- [事实] Docker 一键方式：`make docker-up`（`Makefile` + `deploy/docker-compose.yaml`）。
